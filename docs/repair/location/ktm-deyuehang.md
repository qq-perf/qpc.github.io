# KTMR2R（德悦行店）

::: tip 点评

2020年12月KTM大德良驹闭店之后，德悦行KTMR2R店成为目前北京唯一一家KTMR2R 4S店铺。

售后技师推荐：欧国海，人称海哥。

:::
